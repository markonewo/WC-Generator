WordCloud Generator

Hey. This is a WordCloud Generator built using Python libraries such as Flask and Wordcloud. The deployment of this Flask is completely done using Herokuapp. Heroku is a cloud platform that allows programmers to create free account and deploy their machine learning or data science models on web application.
Entire working of the project is available on my Github profile. Visit the profile using the contacts provided. Connect with me for collaboration or any doubts.
